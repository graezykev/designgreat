# Large Dialog

A spacious dialog for complex content or detailed forms.

## Basic Example

export function LargeDialogExample() {
  const [open, setOpen] = useState(false)
  return (
    <>
      <Button onClick={() => setOpen(true)}>Open Large Dialog</Button>
      <Dialog open={open} onClose={() => setOpen(false)} size="lg">
        <DialogHeader>
          <DialogTitle>Advanced Settings</DialogTitle>
        </DialogHeader>
        <DialogBody>
          <DialogDescription>
            Configure advanced options for your application. These settings may affect performance and behavior.
          </DialogDescription>
          <div style={{ marginTop: '1rem' }}>
            <p>Larger form with multiple sections...</p>
            <p>More content here...</p>
            <p>Additional configuration options...</p>
          </div>
        </DialogBody>
        <DialogFooter>
          <Button variant="outline" onClick={() => setOpen(false)}>
            Cancel
          </Button>
          <Button variant="primary" onClick={() => setOpen(false)}>
            Apply Settings
          </Button>
        </DialogFooter>
      </Dialog>
    </>
  )
}

## Usage Guidelines

### When to Use
- Complex forms (6+ fields)
- Rich content displays
- Data tables or lists within dialogs
- Settings panels with multiple sections

### Best Practices
- Use sparingly - consider full pages instead
- Organize content into clear sections
- Ensure adequate spacing between elements
- Consider breaking into multi-step flows
- Test on smaller screens
