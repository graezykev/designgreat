# Primary

The primary button variant is used for the main call-to-action on a page.

## Usage Guidelines

### When to Use
- Primary action on a page or dialog
- Submit buttons in forms
- Most important action in a group

### Best Practices
- Use only one primary button per section
- Label should be action-oriented (e.g., "Save Changes", "Continue")
- Keep labels concise (2-3 words max)
