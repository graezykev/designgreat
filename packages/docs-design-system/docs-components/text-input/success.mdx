## Usage Guidelines

### When to Use
- Successful form validation
- Real-time validation confirmation
- Account verification status
- Availability checks (username, email)

### Best Practices
- Show success state after validation passes
- Use green color for clear visual confirmation
- Include checkmark icon for additional clarity
- Keep success messages brief and positive
- Don't overuse - reserve for important confirmations
