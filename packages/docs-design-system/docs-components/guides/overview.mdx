# Web Components Guide

`@designgreat/lib-web-ui` is Designgreat's React component library built with design tokens, Tailwind CSS utilities, and custom component styles. This guide explains how to consume the library in your application.

## What's Included

The library provides:

- **React Components:** Button, TextInput, Dialog with full TypeScript support
- **Design Tokens:** CSS variables from `@designgreat/lib-web-ui-design-token`
- **Tailwind Utilities:** Token-aware utility classes for custom UIs
- **Component Styles:** Pre-built styles with theme support
- **TypeScript Definitions:** Fully typed component props

## Features

- ✅ Built-in theme support (light/dark)
- ✅ Accessibility features
- ✅ TypeScript definitions
- ✅ Tailwind CSS integration
- ✅ Design token utilities
- ✅ Framework agnostic (works with Next.js, Vite, CRA, etc.)
