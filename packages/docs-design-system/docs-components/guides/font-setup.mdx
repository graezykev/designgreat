# Font Setup

The library does **not** include `@font-face` declarations in the distributable CSS. This gives you flexibility in how you load fonts. If you want to use the Roboto font family (as defined in the design tokens), you have two options:

## Option 1: Import from Design Token Package

Import the font CSS and font files from the design token package:

```tsx
// In your app entry point (e.g., main.tsx, _app.tsx, app/layout.tsx)
import '@designgreat/lib-web-ui-design-token/font'
import '@designgreat/lib-web-ui/dist/lib-web-ui.css'
```

This is the recommended approach as it ensures font files are versioned with the design tokens.

## Option 2: Use Your Own Font Loading

If you prefer to load fonts differently (CDN, self-hosted, etc.):

```tsx
// Only import the component CSS
import '@designgreat/lib-web-ui/dist/lib-web-ui.css'
```

Then load Roboto (or your preferred font) using your own method:

### Via Google Fonts CDN

```html
<!-- Add to your HTML head -->
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;500;700&display=swap"
  rel="stylesheet"
/>
```

### Self-Hosted Fonts

```css
/* In your global CSS file */
@font-face {
  font-family: 'Roboto';
  src: url('/fonts/Roboto-Regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

/* Add more @font-face rules for other weights/styles */
```

## Using a Different Font

If you want to use a different font family, simply load your preferred font and update the CSS variable:

```css
:root {
  --dg-typography-font-family-base: 'Your Font', sans-serif;
}
```

## Font Weights Used

The design system uses these Roboto weights:

- 300 (Light)
- 400 (Regular)
- 500 (Medium)
- 700 (Bold)

Make sure to load all weights if using a CDN or self-hosting.
