# Theming

The library includes built-in support for light and dark themes through CSS variables and theme classes.

## Default Theme

The light theme attaches to `:root` by default, so all components use light theme colors without additional configuration.

## Available Themes

- `light` – Default light theme
- `dark` – Dark theme

## Theme Utilities

The design token package provides helper functions for theme management:

```tsx
import {
  getThemeClassName,
  applyTheme,
  isThemeApplied,
  getOppositeTheme,
  THEME_CLASSES,
  THEME_SELECTORS
} from '@designgreat/lib-web-ui-design-token'

// Get CSS class name for a theme
getThemeClassName('dark')  // 'dg-theme-dark'
getThemeClassName('light') // '' (light uses :root)

// Apply theme to an element
applyTheme(document.documentElement, 'dark')

// Check if a theme is applied
isThemeApplied(document.documentElement, 'dark') // true/false

// Get the opposite theme
getOppositeTheme('dark') // 'light'

// Direct access to class names
THEME_CLASSES.dark  // 'dg-theme-dark'
THEME_CLASSES.light // ''
```

## Applying a Theme

Apply a theme by wrapping your content with the theme class:

```tsx
import { getThemeClassName } from '@designgreat/lib-web-ui-design-token'

export function App() {
  return (
    <div className={getThemeClassName('dark')}>
      {/* Application content */}
    </div>
  )
}
```

## Dynamic Theme Switching

For runtime theme switching, use state to manage the active theme:

```tsx
import { useState } from 'react'
import { getThemeClassName, type ThemeName } from '@designgreat/lib-web-ui-design-token'

export function App() {
  const [theme, setTheme] = useState<ThemeName>('light')

  const toggleTheme = () => {
    setTheme(current => current === 'dark' ? 'light' : 'dark')
  }

  return (
    <div className={getThemeClassName(theme)}>
      <button onClick={toggleTheme}>Toggle Theme</button>
      {/* Application content */}
    </div>
  )
}
```

## Programmatic Theme Application

For applying themes directly to the DOM:

```tsx
import { applyTheme } from '@designgreat/lib-web-ui-design-token'

// Apply dark theme to the document
applyTheme(document.documentElement, 'dark')

// Switch to light theme
applyTheme(document.documentElement, 'light')
```

## Theme Classes Reference

| Theme | Class Name | CSS Selector |
|-------|------------|--------------|
| Light | (none) | `:root` |
| Dark | `dg-theme-dark` | `.dg-theme-dark` |

## How Themes Work

Themes work by switching CSS variable values. All components use CSS variables like `var(--dg-color-background-button-default)`, which automatically update when a theme class is applied.

For more details on design tokens and theme architecture, see the [Design Tokens Guide](/docs/tutorial/overview).

## Troubleshooting

If themes aren't working, see the [Troubleshooting Guide](./troubleshooting.mdx#theme-isnt-applied).
