# Elevation

Elevation tokens define shadow depths for creating visual hierarchy. They help users understand which elements are closer to them and which are further away.

## Elevation Scale

| Token                                                 | CSS Variable                                               | Shadow          | Use Case            |
| ----------------------------------------------------- | ---------------------------------------------------------- | --------------- | ------------------- |
| <CopyableCode>shadow.elevation.none</CopyableCode>    | <CopyableCode>--dg-shadow-elevation-none</CopyableCode>    | None            | Flat elements       |
| <CopyableCode>shadow.elevation.sm</CopyableCode>      | <CopyableCode>--dg-shadow-elevation-sm</CopyableCode>      | 0 1px 2px       | Subtle lift         |
| <CopyableCode bold>shadow.elevation.md</CopyableCode> | <CopyableCode bold>--dg-shadow-elevation-md</CopyableCode> | 0 4px 6px 1px   | Cards, containers   |
| <CopyableCode>shadow.elevation.lg</CopyableCode>      | <CopyableCode>--dg-shadow-elevation-lg</CopyableCode>      | 0 10px 16px 2px | Dropdowns, popovers |
| <CopyableCode>shadow.elevation.xl</CopyableCode>      | <CopyableCode>--dg-shadow-elevation-xl</CopyableCode>      | 0 20px 24px 4px | Modals, dialogs     |

## Special Shadows

| Token                                        | CSS Variable                                      | Shadow          | Use Case       |
| -------------------------------------------- | ------------------------------------------------- | --------------- | -------------- |
| <CopyableCode>shadow.focus</CopyableCode>    | <CopyableCode>--dg-shadow-focus</CopyableCode>    | 0 0 0 2px       | Focus rings    |
| <CopyableCode>shadow.inset.sm</CopyableCode> | <CopyableCode>--dg-shadow-inset-sm</CopyableCode> | inset 0 1px 2px | Pressed states |
| <CopyableCode>shadow.inset.md</CopyableCode> | <CopyableCode>--dg-shadow-inset-md</CopyableCode> | inset 0 2px 4px | Input fields   |

## Elevation Guidelines

| Level | Shadow | Components                      |
| ----- | ------ | ------------------------------- |
| 0     | None   | Flat backgrounds, dividers      |
| 1     | `sm`   | Buttons (resting), cards (flat) |
| 2     | `md`   | Cards, containers, list items   |
| 3     | `lg`   | Dropdowns, popovers, tooltips   |
| 4     | `xl`   | Modals, dialogs, bottom sheets  |

## Usage Examples

### TypeScript/JavaScript

```typescript
import { tokens } from '@designgreat/lib-design-token'

// Elevation levels map to shadow tokens
const elevations = {
  none: tokens.shadow.elevation.none,
  sm: tokens.shadow.elevation.sm,
  md: tokens.shadow.elevation.md,
  lg: tokens.shadow.elevation.lg,
  xl: tokens.shadow.elevation.xl
}

// Component example
function Card({ elevation = 'md', children }) {
  return (
    <div style={{ boxShadow: `var(--dg-shadow-elevation-${elevation})` }}>
      {children}
    </div>
  )
}
```

## Usage Guidelines

### ✅ Do

- Use elevation to indicate interactivity
- Increase elevation on hover for clickable elements
- Use `focus` shadow for keyboard focus states
- Keep elevation consistent within component types

### ❌ Don't

- Use high elevation for static content
- Mix different elevation levels randomly
- Skip elevation levels (e.g., sm → xl)
- Use elevation as the only interactive indicator

## Technical Details

### File Structure

```
packages/lib-design-token/src/tokens/
  └── shadow.js  # All shadow/elevation tokens
```

### Token Structure

```javascript
{
  shadow: {
    // Elevation scale → --dg-shadow-elevation-*
    elevation: {
      none: { type: 'shadow', value: [{ offsetY: 0, blur: 0, color: 'transparent' }] },
      sm: { type: 'shadow', value: [{ offsetY: '{spacing.px1}', blur: '{spacing.px2}', color: '{color.shadow.DEFAULT}' }] },
      md: { type: 'shadow', value: [{ offsetY: '{spacing.px4}', blur: '{spacing.px6}', color: '{color.shadow.DEFAULT}' }] },
      lg: { type: 'shadow', value: [{ offsetY: '{spacing.px10}', blur: '{spacing.px16}', color: '{color.shadow.DEFAULT}' }] },
      xl: { type: 'shadow', value: [{ offsetY: '{spacing.px20}', blur: '{spacing.px24}', color: '{color.shadow.DEFAULT}' }] }
    },

    // Focus ring → --dg-shadow-focus
    focus: { type: 'shadow', value: [{ blur: 0, spread: '{spacing.px2}', color: '{color.primary.DEFAULT}' }] },

    // Inset shadows → --dg-shadow-inset-*
    inset: {
      sm: { type: 'shadow', value: [{ offsetY: '{spacing.px1}', blur: '{spacing.px2}', color: '{color.shadow.DEFAULT}', type: 'inset' }] },
      md: { type: 'shadow', value: [{ offsetY: '{spacing.px2}', blur: '{spacing.px4}', color: '{color.shadow.DEFAULT}', type: 'inset' }] }
    }
  }
}
```
