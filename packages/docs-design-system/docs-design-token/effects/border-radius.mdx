# Border Radius

Border radius tokens define corner rounding for UI elements. They provide a consistent scale from sharp corners to fully rounded shapes.

## Core Scale

| Token | CSS Variable | Value | Pixels |
|-------|--------------|-------|--------|
| <CopyableCode>size.border-radius.none</CopyableCode> | <CopyableCode>--dg-size-border-radius-none</CopyableCode> | 0 | 0px |
| <CopyableCode>size.border-radius.sm</CopyableCode> | <CopyableCode>--dg-size-border-radius-sm</CopyableCode> | 0.125rem | 2px |
| <CopyableCode bold>size.border-radius.DEFAULT</CopyableCode> | <CopyableCode bold>--dg-size-border-radius-default</CopyableCode> | 0.25rem | 4px |
| <CopyableCode>size.border-radius.md</CopyableCode> | <CopyableCode>--dg-size-border-radius-md</CopyableCode> | 0.375rem | 6px |
| <CopyableCode>size.border-radius.lg</CopyableCode> | <CopyableCode>--dg-size-border-radius-lg</CopyableCode> | 0.5rem | 8px |
| <CopyableCode>size.border-radius.xl</CopyableCode> | <CopyableCode>--dg-size-border-radius-xl</CopyableCode> | 0.75rem | 12px |
| <CopyableCode>size.border-radius.2xl</CopyableCode> | <CopyableCode>--dg-size-border-radius-2xl</CopyableCode> | 1rem | 16px |
| <CopyableCode>size.border-radius.3xl</CopyableCode> | <CopyableCode>--dg-size-border-radius-3xl</CopyableCode> | 1.5rem | 24px |
| <CopyableCode>size.border-radius.full</CopyableCode> | <CopyableCode>--dg-size-border-radius-full</CopyableCode> | 9999rem | Pill |

## Usage Guidelines

### ✅ Do

- Use semantic tokens (`button`, `card`, `modal`) for consistency
- Use `full` for pills, badges, and avatars
- Match radius to element size (larger elements = larger radius)
- Keep radius consistent within component families

### ❌ Don't

- Use large radius on small elements (looks unbalanced)
- Mix different radius values on the same component
- Use `full` on rectangular elements (unless intentional pill shape)
- Create new radius values outside the token system

## Technical Details

### File Structure

```
packages/lib-design-token/src/tokens/size/
  └── border-radius.js  # All border radius tokens
```

### Token Structure

```javascript
{
  size: {
    'border-radius': {
      none: { value: 0, type: 'dimension' },
      sm: { value: '{spacing.px2}', type: 'dimension' },
      DEFAULT: { value: '{spacing.px4}', type: 'dimension' },
      md: { value: '{spacing.px6}', type: 'dimension' },
      lg: { value: '{spacing.px8}', type: 'dimension' },
      xl: { value: '{spacing.px12}', type: 'dimension' },
      '2xl': { value: '{spacing.px16}', type: 'dimension' },
      '3xl': { value: '{spacing.px24}', type: 'dimension' },
      full: { value: 9999, type: 'dimension' },
      // Semantic mappings
      button: { value: '{size.border-radius.md}', type: 'dimension' },
      card: { value: '{size.border-radius.lg}', type: 'dimension' },
      // ...
    }
  }
}
```
