# Gradients

Gradient tokens provide pre-defined linear gradients for brand elements, backgrounds, and decorative effects. They ensure consistent gradient usage across your application.

## Available Gradients

| Token                                                    | CSS Variable                                              | Direction | Use Case               |
| -------------------------------------------------------- | --------------------------------------------------------- | --------- | ---------------------- |
| <CopyableCode bold>gradient.brand-primary</CopyableCode> | <CopyableCode>--dg-gradient-brand-primary</CopyableCode>  | 135¬∞      | CTAs, brand highlights |
| <CopyableCode>gradient.surface-subtle</CopyableCode>     | <CopyableCode>--dg-gradient-surface-subtle</CopyableCode> | 180¬∞      | Subtle backgrounds     |
| <CopyableCode>gradient.hero</CopyableCode>               | <CopyableCode>--dg-gradient-hero</CopyableCode>           | 135¬∞      | Hero sections          |
| <CopyableCode>gradient.accent</CopyableCode>             | <CopyableCode>--dg-gradient-accent</CopyableCode>         | 90¬∞       | Badges, highlights     |
| <CopyableCode>gradient.overlay-dark</CopyableCode>       | <CopyableCode>--dg-gradient-overlay-dark</CopyableCode>   | 180¬∞      | Text on images         |

## Usage Guidelines

### ‚úÖ Do

- Use `brand-primary` for important CTAs
- Use `hero` for large decorative sections
- Use `overlay-dark` to improve text legibility on images
- Keep gradient usage minimal and purposeful
- Add hover transitions for interactive elements

### ‚ùå Don't

- Overuse gradients (they should be special)
- Use gradients for small text elements (except headlines)
- Create custom gradients outside the token system
- Use gradients where solid colors work fine
- Apply gradient text to body copy
