# Transitions

Transition tokens provide pre-built transition presets that combine property, duration, and easing for common use cases. They simplify applying consistent animations across your application.

## Transition Presets

| Token | CSS Variable | Properties | Duration | Easing |
|-------|--------------|------------|----------|--------|
| <CopyableCode bold>transition.all</CopyableCode> | <CopyableCode>--dg-transition-all</CopyableCode> | `all` | 200ms | ease-out |
| <CopyableCode>transition.colors</CopyableCode> | <CopyableCode>--dg-transition-colors</CopyableCode> | `color, background-color, border-color` | 100ms | ease |
| <CopyableCode>transition.opacity</CopyableCode> | <CopyableCode>--dg-transition-opacity</CopyableCode> | `opacity` | 200ms | ease-in-out |
| <CopyableCode>transition.transform</CopyableCode> | <CopyableCode>--dg-transition-transform</CopyableCode> | `transform` | 200ms | ease-out |

## Combining Transitions

You can combine multiple transition presets for fine-grained control:

```css
.interactive-card {
  transition: 
    var(--dg-transition-colors),
    var(--dg-transition-transform),
    var(--dg-transition-shadow);
}
```

Or use `transition.all` for simplicity when multiple properties change:

```css
.interactive-card {
  transition: var(--dg-transition-all);
}
```

## Usage Guidelines

### ✅ Do

- Use `transition.colors` for simple color changes (faster)
- Use `transition.all` when multiple properties change
- Use `transition.slow` for modals and emphasis effects
- Use `transition.slower` for complex animations and dramatic reveals
- Use `transition.none` to explicitly disable transitions

### ❌ Don't

- Use `transition.all` when only colors change (less performant)
- Apply transitions to properties that don't need them
- Make transitions too slow (users will notice lag)
- Forget to test transitions on lower-end devices

```javascript

  transition: {
    all: {
      type: 'transition',
      value: {
        property: 'all',
        duration: '{duration.normal}',
        timingFunction: '{cubic-bezier.ease-out}',
        delay: 0
      }
    },
    slow: {
      type: 'transition',
      value: {
        property: 'all',
        duration: '{duration.slow}',
        timingFunction: '{cubic-bezier.ease-in-out}',
        delay: 0
      }
    },
    slower: {
      type: 'transition',
      value: {
        property: 'all',
        duration: '{duration.slower}',
        timingFunction: '{cubic-bezier.ease-in-out}',
        delay: 0
      }
    },
    // ...
  }
}
```
