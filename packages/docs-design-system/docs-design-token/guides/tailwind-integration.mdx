# Tailwind CSS Integration

Tailwind CSS v4 favors authoring theme data in CSS. You can import the design token CSS directly alongside the Tailwind entry point.

## Import Design Token CSS

Simply import the combined CSS that includes both light and dark themes:

```css
/* src/styles/tailwind.css */
@import 'tailwindcss';
@import '@designgreat/lib-web-ui-design-token/css';

@custom-variant theme-dark (&:where(.dg-theme-dark, .dg-theme-dark *));
```

This imports:
- `:root` with light theme variables
- `.dg-theme-dark` with dark theme variables

## Using Design Tokens in Utilities

With the CSS imported, utilities can directly reference token values:

```tsx
<div className="bg-[var(--dg-color-background-default)] text-[var(--dg-color-text-default)]">
  Content
</div>
```

## Custom Variant Example

The `@custom-variant` allows you to create responsive dark mode utilities:

```tsx
<div className="bg-white theme-dark:bg-gray-900">
  Adapts to theme
</div>
```

## Alternative: Tailwind v3 Configuration

For Tailwind v3, you can extend the theme in your config:

```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        // Reference design token variables
        'dg-primary': 'var(--dg-color-primary-default)',
        'dg-secondary': 'var(--dg-color-secondary-default)',
        'dg-background': 'var(--dg-color-background-default)',
        'dg-text': 'var(--dg-color-text-default)'
      },
      spacing: {
        // Reference spacing tokens
        'dg-1': 'var(--dg-spacing-1)',
        'dg-2': 'var(--dg-spacing-2)',
        'dg-4': 'var(--dg-spacing-4)',
        'dg-8': 'var(--dg-spacing-8)',
        'dg-11': 'var(--dg-spacing-11)'
      }
    }
  }
}
```

Then use them as regular Tailwind utilities:

```tsx
<div className="bg-dg-background text-dg-text p-dg-11">
  Content with design tokens
</div>
```

## JavaScript-Driven Theming

You can also access tokens programmatically and use theme utilities:

```typescript
import {
  themes,
  applyTheme,
  getThemeClassName,
  THEME_CLASSES
} from '@designgreat/lib-web-ui-design-token'

// Access token values
const darkTokens = themes.dark
const bgColor = darkTokens.dg.color.background.DEFAULT

// Apply theme using utility function
applyTheme(document.documentElement, 'dark')

// Or use the class name directly
document.body.className = getThemeClassName('dark')

// Or access the constant
document.body.className = THEME_CLASSES.dark
```

## Benefits

- ✅ Design tokens work with Tailwind utilities
- ✅ Automatic theme switching support
- ✅ Type-safe token access in JavaScript
- ✅ Compatible with both Tailwind v3 and v4
