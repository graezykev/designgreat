# Basic Usage

:::tip TL;DR
Import CSS → use `var(--dg-*)` in your styles → colors automatically adapt to light/dark theme.
:::

This guide covers the most common ways to use design tokens in your application.

---

## Using CSS Variables

After [installing](./installation) and importing the CSS, reference tokens in your styles:

```css
.card {
  /* Colors */
  background-color: var(--dg-color-background-default);
  color: var(--dg-color-text-default);
  border: 1px solid var(--dg-color-border-default);
  
  /* Spacing */
  padding: var(--dg-spacing-px16);
  margin-bottom: var(--dg-spacing-px24);
  
  /* Typography */
  font-family: var(--dg-font-family-primary);
}

.card-title {
  color: var(--dg-color-text-bold);
  font-weight: var(--dg-font-weight-bold);
}

.card-link {
  color: var(--dg-color-primary-DEFAULT);
}

.card-link:hover {
  color: var(--dg-color-primary-hover-DEFAULT);
}
```

## Theme-Aware Styling

Design tokens automatically adapt to the current theme. You don't need conditional logic:

```css
/* This SAME code works in both light and dark themes */
.notification {
  background: var(--dg-color-background-information-DEFAULT);
  color: var(--dg-color-text-information-DEFAULT);
  border: 1px solid var(--dg-color-border-information);
}
```

When `.dg-theme-dark` is applied to an ancestor, these variables automatically resolve to dark-appropriate values.

:::tip Deep Dive
To understand which colors change between themes and why, see [Core Concepts & Theme-Awareness](/design-token/colors/core-concepts-theme-awareness).
:::

## SCSS Usage

For SCSS projects, import the SCSS variables:

```scss
@import '@designgreat/lib-design-token/scss/light';

.my-component {
  background: $dg-color-background-default;
  color: $dg-color-text-default;
  padding: $dg-spacing-px16;
}
```

:::warning
SCSS variables are **not theme-aware** — they compile to static values. For theme switching, use CSS variables instead.
:::

## JavaScript/TypeScript Access

Access token values programmatically when needed:

```typescript
import { themes } from '@designgreat/lib-design-token'

// Access any token value
const primaryColor = themes.light.dg.color.primary.DEFAULT  // "#0055cc"
const spacing = themes.light.dg.spacing.px16                 // "1rem"

// Compare light vs dark
const lightBg = themes.light.dg.color.background.DEFAULT    // "#ffffff"
const darkBg = themes.dark.dg.color.background.DEFAULT      // "#0c0c0d"
```

For full API documentation, see [API Reference](./api/reference).
