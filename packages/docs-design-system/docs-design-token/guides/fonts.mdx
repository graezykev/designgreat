# Fonts

:::tip TL;DR
`import '@designgreat/lib-design-token/font'` — that's it. Roboto is now available.
:::

The design token package includes **Roboto** font files so you don't need a CDN or external dependencies.

---

## Quick Start

Import fonts in your app's entry file:

```tsx
// main.tsx, app/layout.tsx, or _app.tsx
import '@designgreat/lib-design-token/font'
import '@designgreat/lib-design-token/css'
```

Use in your styles:

```css
body {
  font-family: var(--dg-font-family-primary);
}

.heading {
  font-weight: 700; /* Bold */
}

.body-text {
  font-weight: 400; /* Regular */
}
```

---

## Alternative: Google Fonts CDN

If you prefer not to bundle fonts, load Roboto from Google Fonts instead:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link
  href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;500;700&display=swap"
  rel="stylesheet"
/>
```

Then skip the font import:

```tsx
// No font import needed
import '@designgreat/lib-design-token/css'
```

---

## Available Weights

| Weight | Name | Use Case |
|--------|------|----------|
| 100 | Thin | Decorative, large display text |
| 300 | Light | Subtitles, captions |
| 400 | Regular | Body text (default) |
| 500 | Medium | Emphasized text, subheadings |
| 700 | Bold | Headings, strong emphasis |
| 900 | Black | Display text, hero headers |

Each weight includes both **normal** and **italic** styles.

## Framework Examples

### React (Vite)

```tsx
// src/main.tsx
import '@designgreat/lib-design-token/font'
import '@designgreat/lib-design-token/css'
import { createRoot } from 'react-dom/client'
import App from './App'

createRoot(document.getElementById('root')!).render(<App />)
```

### Next.js (App Router)

```tsx
// app/layout.tsx
import '@designgreat/lib-design-token/font'
import '@designgreat/lib-design-token/css'

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

### Next.js (Pages Router)

```tsx
// pages/_app.tsx
import '@designgreat/lib-design-token/font'
import '@designgreat/lib-design-token/css'
import type { AppProps } from 'next/app'

export default function App({ Component, pageProps }: AppProps) {
  return <Component {...pageProps} />
}
```

### Remix

```tsx
// app/root.tsx
import '@designgreat/lib-design-token/font'
import '@designgreat/lib-design-token/css'
import { Links, Meta, Outlet, Scripts } from '@remix-run/react'

export default function App() {
  return (
    <html lang="en">
      <head>
        <Meta />
        <Links />
      </head>
      <body>
        <Outlet />
        <Scripts />
      </body>
    </html>
  )
}
```

## Using a Different Font

<details>
<summary>**Skip Roboto and use your own font**</summary>

### Step 1: Don't Import the Font Package

```tsx
// Skip this import
// import '@designgreat/lib-design-token/font'

// Only import tokens
import '@designgreat/lib-design-token/css'
```

### Step 2: Load Your Font

**Via Google Fonts:**

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link 
  href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap"
  rel="stylesheet"
/>
```

**Or self-hosted:**

```css
@font-face {
  font-family: 'Inter';
  src: url('/fonts/Inter-Regular.woff2') format('woff2');
  font-weight: 400;
  font-display: swap;
}
```

### Step 3: Override the CSS Variable

```css
:root,
.dg-theme-dark {
  --dg-font-family-primary: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
}
```

</details>

## Static Site Generators

For **Docusaurus, Astro**, and similar SSGs that can't process CSS font imports at build time, use the CLI to copy fonts:

```bash
npx @designgreat/lib-design-token dg-copy-fonts ./static/font
```

Add to your scripts:

```json
{
  "scripts": {
    "copy:fonts": "dg-copy-fonts ./static/font",
    "dev": "npm run copy:fonts && your-dev-command",
    "build": "npm run copy:fonts && your-build-command"
  }
}
```

Then reference in your site config:

```typescript
// docusaurus.config.ts
export default {
  stylesheets: [`${baseUrl}font/font-face.css`]
}
```

## Troubleshooting

<details>
<summary>**Common issues**</summary>

### Fonts Don't Load

1. ✅ Font import comes **before** component imports
2. ✅ Check Network tab for `.woff2` files loading
3. ✅ No console errors
4. ✅ Package is installed correctly

### Flash of Unstyled Text (FOUT)

This is expected — the package uses `font-display: swap` to prevent invisible text.

To minimize:
```html
<link 
  rel="preload" 
  href="/fonts/Roboto-Regular.woff2" 
  as="font" 
  type="font/woff2" 
  crossorigin 
/>
```

### 404 Errors in Production (SSG)

For static site generators, ensure fonts are copied to your output directory before build. See the copy script above.

</details>
