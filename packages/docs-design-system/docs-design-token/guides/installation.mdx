# Installation

:::tip TL;DR
`pnpm add @designgreat/lib-design-token` → import CSS + fonts → use `var(--dg-*)` variables.
:::

## Install the Package

```bash
pnpm add @designgreat/lib-design-token
```

Or with npm/yarn:

```bash
npm install @designgreat/lib-design-token
yarn add @designgreat/lib-design-token
```

:::tip Using React Components?
If you want pre-built React components (Button, Dialog, TextInput), see the [Web Component Installation Guide](/web-component/guides/installation) instead. That package includes design tokens automatically.
:::

---

## What You Get

The package provides three main exports:

| Export | What It Provides | Import |
|--------|------------------|--------|
| **CSS Variables** | Light & dark theme tokens | `@import '@.../css'` |
| **Fonts** | Roboto font files + @font-face | `import '@.../font'` |
| **TypeScript APIs** | Runtime token access & theme utilities | `import { ... } from '@...'` |

---

## Recommended Setup

Add these imports to your app's entry file (e.g., `main.tsx`, `app/layout.tsx`):

```tsx
// 1. Import fonts (optional but recommended)
import '@designgreat/lib-design-token/font'

// 2. Import CSS variables (required)
import '@designgreat/lib-design-token/css'
```

That's the minimum setup! Now you can use design tokens in your CSS:

```css
.my-element {
  background: var(--dg-color-background-default);
  color: var(--dg-color-text-default);
}
```

## Package Exports Reference

<details>
<summary>**View all available exports**</summary>

### CSS Exports

```css
/* Combined: both light (:root) and dark (.dg-theme-dark) themes */
@import '@designgreat/lib-design-token/css';

/* Or import themes separately */
@import '@designgreat/lib-design-token/css/light';
@import '@designgreat/lib-design-token/css/dark';

/* SCSS variables (theme-specific) */
@import '@designgreat/lib-design-token/scss/light';
@import '@designgreat/lib-design-token/scss/dark';
```

### Font Export

```typescript
// Includes @font-face definitions + bundled .woff2 files
import '@designgreat/lib-design-token/font'
```

### TypeScript/JavaScript Exports

```typescript
import {
  // Theme utilities
  getThemeClassName,    // Get CSS class name for a theme
  getThemeSelector,     // Get CSS selector for a theme
  applyTheme,           // Apply theme to an element
  isThemeApplied,       // Check if theme is active
  getOppositeTheme,     // Get opposite theme name
  THEME_CLASSES,        // { light: '', dark: 'dg-theme-dark' }
  THEME_SELECTORS,      // { light: ':root', dark: '.dg-theme-dark' }
  
  // Token access
  themes,               // { light: {...}, dark: {...} }
  light,                // Light theme tokens
  dark,                 // Dark theme tokens
  createTheme,          // Create custom theme with overrides
  getThemeTokens,       // Get tokens for a specific theme
  listThemeNames,       // Get array of theme names
  
  // Types
  type ThemeName,       // 'light' | 'dark'
  type ThemeToken,      // Full token structure
  type ThemeOverrides   // Partial structure for overrides
} from '@designgreat/lib-design-token'
```
