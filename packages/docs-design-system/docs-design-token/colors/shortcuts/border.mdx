# Border Shortcuts

Border tokens provide consistent, theme-aware borders across your application. The system includes both complete border shorthand tokens and individual border color tokens.

## Overview

The design system includes pre-defined border tokens that combine width, style, and color into ready-to-use values. These tokens automatically adapt between light and dark themes.

### Key Characteristics

- **Theme-aware**: Border colors adapt to light/dark mode
- **Complete shorthand**: Tokens include width, style, and color
- **Semantic support**: Special borders for inputs, dividers, and focus states
- **Consistent widths**: Standardized width scale from thin to thick

## Border Tokens

### Complete Border Shorthands

These tokens provide complete `border` values (width + style + color):

| Token | CSS Variable | Use Case |
|-------|--------------|----------|
| `border.default` | `--dg-border-default` | Standard border for cards, containers |
| `border.subtle` | `--dg-border-subtle` | Light separation, subtle dividers |
| `border.bold` | `--dg-border-bold` | Emphasized borders |

### Border Width Tokens

Separate width tokens for custom border compositions:

| Token | CSS Variable | Value | Use Case |
|-------|--------------|-------|----------|
| `border.width.none` | `--dg-border-width-none` | 0 | No border |
| `border.width.thin` | `--dg-border-width-thin` | 1px | Default borders |
| `border.width.medium` | `--dg-border-width-medium` | 2px | Emphasized borders, focus rings |
| `border.width.thick` | `--dg-border-width-thick` | 4px | Heavy emphasis, accent borders |

## Usage Guidelines

### ✅ Do

- Use `--dg-border-default` for most container borders
- Use `--dg-border-input-*` tokens for form elements
- Use `--dg-border-divider-*` for horizontal separators
- Use `--dg-border-focus` alongside `--dg-shadow-focus` for accessibility
- Test border visibility in both light and dark themes

### ❌ Don't

- Create custom border colors outside the token system
- Use hardcoded pixel values for border widths
- Mix border patterns inconsistently
- Use semantic border colors for non-semantic elements
- Forget to consider hover and focus states

## Accessibility

### Focus Indicators

Always provide visible focus indicators:

```css
.interactive-element:focus {
  outline: var(--dg-border-focus);
  outline-offset: 2px;
  box-shadow: var(--dg-shadow-focus);
}
```

### Contrast Requirements

- UI components and borders: Minimum 3:1 contrast ratio
- Consider using `--dg-border-bold` for better visibility
- Test border visibility in both light and dark themes

## Technical Details

### Token Structure

```
border/
├── width/
│   ├── none        → 0
│   ├── thin        → 1px
│   ├── medium      → 2px
│   └── thick       → 4px
├── default         → thin solid border-color
├── subtle          → thin solid subtle-color
├── bold            → thin solid bold-color
├── input/
│   ├── DEFAULT     → thin solid input-color
│   ├── focus       → thin solid primary
│   ├── error       → thin solid error
│   └── success     → thin solid success
├── divider/
│   ├── DEFAULT     → thin solid subtle-color
│   └── bold        → medium solid default-color
└── focus           → medium solid primary
```

### File Structure

```
packages/lib-design-token/src/tokens/
├── border.js              # Border tokens (shorthand, width, input, divider)
└── color/shortcut/
    └── border.js          # Border color shortcuts
```

### CSS Variables Reference

```css
/* Border widths */
--dg-border-width-none: 0;
--dg-border-width-thin: 1px;
--dg-border-width-medium: 2px;
--dg-border-width-thick: 4px;

/* Complete border shorthands */
--dg-border-default: 1px solid var(--dg-color-border-default);
--dg-border-subtle: 1px solid var(--dg-color-border-subtle);
--dg-border-bold: 1px solid var(--dg-color-border-bold);

/* Input borders */
--dg-border-input-default: 1px solid var(--dg-color-border-input-default);
--dg-border-input-focus: 1px solid var(--dg-color-primary-default);
--dg-border-input-error: 1px solid var(--dg-color-semantic-error-default);
--dg-border-input-success: 1px solid var(--dg-color-semantic-success-default);

/* Divider borders */
--dg-border-divider-default: 1px solid var(--dg-color-border-subtle);
--dg-border-divider-bold: 2px solid var(--dg-color-border-default);

/* Focus border */
--dg-border-focus: 2px solid var(--dg-color-primary-default);
```

### TypeScript/JavaScript

```typescript
import { tokens } from '@designgreat/lib-design-token'

// Access border tokens
const borders = {
  default: tokens.border.default,
  subtle: tokens.border.subtle,
  bold: tokens.border.bold,
  inputDefault: tokens.border.input.DEFAULT,
  inputFocus: tokens.border.input.focus,
  divider: tokens.border.divider.DEFAULT,
  focus: tokens.border.focus
}

// Usage in component
function Card({ variant = 'default', children }) {
  return (
    <div style={{ border: `var(--dg-border-${variant})` }}>
      {children}
    </div>
  )
}
```
