## Usage

```css
.element {
  padding: var(--dg-spacing-px16);
  margin-bottom: var(--dg-spacing-px24);
  gap: var(--dg-spacing-px8);
}
```

:::tip
Prefer [semantic scales](./semantic-scales) over pixel values for better maintainability.
:::

## Technical Details

### File Structure

```
packages/lib-design-token/src/tokens/size/
  └── index.js  # All spacing tokens (pixel and semantic)
```

### Token Structure

```javascript
{
  spacing: {
    // Pixel values (rem-based)
    'px0': { type: 'dimension', value: 0 },
    'px1': { type: 'dimension', value: 1 / 16 },  // 0.0625rem
    'px2': { type: 'dimension', value: 2 / 16 },  // 0.125rem
    'px4': { type: 'dimension', value: 4 / 16 },  // 0.25rem
    // ...
  }
}
```
