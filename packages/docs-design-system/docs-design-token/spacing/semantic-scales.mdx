```typescript
  gap: {

    lg: tokens.spacing.gap.lg,

  },
  inset: {
    sm: tokens.spacing.inset.sm,

  },

    sm: tokens.spacing.stack.sm,
    md: tokens.spacing.stack.md,
    lg: tokens.spacing.stack.lg
  }
}

// Component example
function Card({ padding = 'md', children }) {
  return (
    <div style={{ padding: `var(--dg-spacing-inset-${padding})` }}>
      {children}
    </div>
  )
}
```

## Usage Guidelines

### ✅ Do

- Use semantic scales for consistent layouts
- Match scale to use case (`gap` for gaps, `inset` for padding)
- Use `md` as your default starting point

### ❌ Don't

- Mix pixel values with semantic scales
- Use `space` when a more specific scale exists
- Skip sizes (e.g., jumping from `xs` to `xl`)

## Technical Details

### File Structure


### Token Structure

```javascript
{
  spacing: {
    // Semantic scales reference pixel values
    space: {
      none: { value: 0 },
      xs: { value: '{spacing.px4}' },
      sm: { value: '{spacing.px8}' },
      md: { value: '{spacing.px16}' },
      // ...
    },
    gap: { /* Similar structure */ },
    inset: { /* Similar structure */ },
    stack: { /* Similar structure */ }
  }
}
```
