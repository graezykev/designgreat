## Common Use Cases

| Weight | Token       | Use Case                  |
| ------ | ----------- | ------------------------- |
| 400    | `regular`   | Body text, paragraphs     |
| 500    | `medium`    | Subtle emphasis, labels   |
| 600    | `semi-bold` | Subheadings, buttons      |
| 700    | `bold`      | Headings, strong emphasis |

## Usage Examples

### CSS

```css
body {
  font-weight: var(--dg-font-weight-regular);
}

h1 {
  font-weight: var(--dg-font-weight-bold);
}

.button {
  font-weight: var(--dg-font-weight-semi-bold);
}

.label {
  font-weight: var(--dg-font-weight-medium);
}
```

### TypeScript/JavaScript

```typescript
import { tokens } from '@designgreat/lib-design-token'

// Font weight styles
const weights = {
  thin: tokens['font-weight'].thin,
  regular: tokens['font-weight'].regular,
  medium: tokens['font-weight'].medium,
  'semi-bold': tokens['font-weight']['semi-bold'],
  bold: tokens['font-weight'].bold
}

// Component example
function Text({ weight = 'regular', children }) {
  return (
    <p style={{ fontWeight: `var(--dg-font-weight-${weight})` }}>
      {children}
    </p>
  )
}
```

## Usage Guidelines

### ✅ Do

- Use `regular` (400) for body text
- Use `semi-bold` (600) or `bold` (700) for headings
- Limit weight variations per page for visual consistency

### ❌ Don't

- Use thin weights for body text (poor readability)
- Use more than 3-4 weights on a single page
- Mix extreme weights without purpose
