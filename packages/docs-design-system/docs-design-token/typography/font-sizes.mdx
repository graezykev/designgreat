## Usage Examples

### CSS

```css
h1 {
  font-size: var(--dg-size-font-h1);
}

p {
  font-size: var(--dg-size-font-body);
}

.label {
  font-size: var(--dg-size-font-overline);
  text-transform: uppercase;
  letter-spacing: var(--dg-letter-spacing-wider);
}
```

### TypeScript/JavaScript

```typescript
import { tokens } from '@designgreat/lib-design-token'

// Font size styles
const typography = {
  title: tokens.size.font.title,
  h1: tokens.size.font.h1,
  h2: tokens.size.font.h2,
  h3: tokens.size.font.h3,
  body: tokens.size.font.body,
  caption: tokens.size.font.caption
}

// Component example
function Heading({ level = 1, children }) {
  return (
    <h1 style={{ fontSize: `var(--dg-size-font-h${level})` }}>
      {children}
    </h1>
  )
}
```

## Usage Guidelines

### ✅ Do

- Use semantic names (`h1`, `body`) for document structure
- Use t-shirt sizes (`sm`, `lg`) for component variations
- Pair with appropriate line heights

### ❌ Don't

- Skip heading levels (h1 → h3)
- Use raw pixel values
- Mix semantic and t-shirt sizes inconsistently

## Technical Details

### File Structure

```
packages/lib-design-token/src/tokens/
  ├── size/font.js           # Generic font sizes (xs–4xl)
  └── typography/font-size.js # Semantic font sizes (title, h1–h6, body)
```

### Token Structure

```javascript
{
  size: {
    font: {
      title: { type: 'fontSize', value: '{spacing.px40}' },
      h1: { type: 'fontSize', value: '{spacing.px32}' },
      h2: { type: 'fontSize', value: '{spacing.px24}' },
      h3: { type: 'fontSize', value: '{spacing.px20}' },
      h4: { type: 'fontSize', value: '{spacing.px16}' },
      h5: { type: 'fontSize', value: '{spacing.px14}' },
      h6: { type: 'fontSize', value: '{spacing.px12}' },
      body: { type: 'fontSize', value: '{spacing.px16}' },
      caption: { type: 'fontSize', value: '{spacing.px12}' },
      overline: { type: 'fontSize', value: '{spacing.px10}' },
      sub: { type: 'fontSize', value: '{spacing.px10}' }
    }
  }
}
```
