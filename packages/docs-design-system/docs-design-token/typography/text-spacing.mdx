# Text Spacing

Tokens for line height, letter spacing, and line clamp.

### Semantic Line Heights

Pre-mapped to text elements:

| Token | Maps To | Use Case |
|-------|---------|----------|
| `line-height.title` | `none` (1) | Display titles |
| `line-height.h1` | `tight` (1.25) | H1 headings |
| `line-height.h2` | `tight` (1.25) | H2 headings |
| `line-height.h3` | `snug` (1.375) | H3 headings |
| `line-height.body` | `normal` (1.5) | Body text |
| `line-height.caption` | `normal` (1.5) | Captions |

| <CopyableCode>letter-spacing.wider</CopyableCode> | <CopyableCode>--dg-letter-spacing-wider</CopyableCode> | 0.05em | <span style={{letterSpacing: 'var(--dg-letter-spacing-wider)', fontSize: 'var(--dg-size-font-xs)', textTransform: 'uppercase'}}>WIDER SPACING</span> |
| <CopyableCode>letter-spacing.widest</CopyableCode> | <CopyableCode>--dg-letter-spacing-widest</CopyableCode> | 0.1em | <span style={{letterSpacing: 'var(--dg-letter-spacing-widest)', fontSize: 'var(--dg-size-font-xs)', textTransform: 'uppercase'}}>WIDEST SPACING</span> |

### Letter Spacing Guidelines

| Context | Recommended | Why |
|---------|-------------|-----|
| Large headings | `tight` or `tighter` | Tighten as size increases |
| Body text | `normal` | Default, no adjustment |
| All caps labels | `wide` or `wider` | Improves readability |
| Small caps | `wider` or `widest` | Compensates for reduced height |

## Usage

```css
/* Line height */
p {
  line-height: var(--dg-line-height-normal);
}

h1 {
  line-height: var(--dg-line-height-tight);
  letter-spacing: var(--dg-letter-spacing-tight);
}

/* Overline label */
.overline {
  text-transform: uppercase;
  letter-spacing: var(--dg-letter-spacing-wider);
}

/* Line clamp */
.card-description {
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: var(--dg-line-clamp-2);
  overflow: hidden;
}
```

## Technical Details

### File Structure

```
packages/lib-design-token/src/tokens/typography/
  ├── line-height.js     # Line height tokens
  ├── letter-spacing.js  # Letter spacing tokens
  └── line-clamp.js      # Line clamp tokens
```

### Token Structure

```javascript
// Line height (unitless values)
{ 'line-height': { tight: { value: 1.25 }, normal: { value: 1.5 }, ... }}

// Letter spacing (em values)
{ 'letter-spacing': { tight: { value: -0.025 }, normal: { value: 0 }, ... }}

// Line clamp (integer values)
{ 'line-clamp': { '1': { value: 1 }, '2': { value: 2 }, ... }}
```
