# Package Exports

The package provides the following exports for easy consumption:

## Main Export

```typescript
import { ... } from '@designgreat/lib-web-ui-design-token'
```

TypeScript runtime APIs including theme management and token access.

## Font Export

```typescript
import '@designgreat/lib-web-ui-design-token/font'
```

Font-face definitions and font files (self-contained). This import includes both the CSS and the actual font files.

## CSS Theme Exports

### Light Theme

```css
@import '@designgreat/lib-web-ui-design-token/css/light';
```

Light theme CSS variables.

### Dark Theme

```css
@import '@designgreat/lib-web-ui-design-token/css/dark';
```

Dark theme CSS variables.

:::warning
Both `light` and `dark` CSS files use `:root` selector. **Do not import both** or they will conflict. For theme switching support, use the `@designgreat/design-token-support` package instead.
:::

## Export Summary

| Export | Purpose | Usage |
|--------|---------|-------|
| Main package | TypeScript APIs | Runtime token access |
| `/font` | Font assets | Font-face + font files |
| `/css/light` | Light theme | Single theme CSS |
| `/css/dark` | Dark theme | Single theme CSS |
