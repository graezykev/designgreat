## Usage Guidelines

### When to Use
- Secondary actions in a group
- Cancel or back buttons
- Alternative actions that complement a primary action

### Best Practices
- Pair with a primary button for clear hierarchy
- Use for actions that are important but not primary
- Maintain consistent styling across the application
