## Usage Guidelines

### When to Use
- Tertiary actions
- Less emphasis needed but still clickable
- On colored backgrounds where solid buttons don't work well
- Filter or toggle buttons

### Best Practices
- Use for the least important actions in a button group
- Ensure sufficient contrast with background
- Good for repetitive actions (like in lists or tables)
