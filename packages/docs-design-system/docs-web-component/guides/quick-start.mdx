# Quick Start

Get a React component working in **2 minutes**.

## 1. Install

```bash
pnpm add @designgreat/lib-web-ui
```

## 2. Import

Add this to your app's entry file:

```tsx
// main.tsx, app/layout.tsx, or _app.tsx
import '@designgreat/lib-web-ui/dist/lib-web-ui.css'
```

## 3. Use

Import and use components:

```tsx
import { Button } from '@designgreat/lib-web-ui'

function App() {
  return (
    <Button variant="primary" onClick={() => alert('It works!')}>
      Click me
    </Button>
  )
}
```

**That's it!** You now have:
- ‚úÖ Styled React components (Button, TextInput, Dialog)
- ‚úÖ Light & dark theme support
- ‚úÖ Design token CSS variables

## Add Dark Mode

Wrap your app and toggle the theme class:

```tsx
import { useState } from 'react'
import { getThemeClassName } from '@designgreat/lib-web-ui-design-token'
import { Button } from '@designgreat/lib-web-ui'

function App() {
  const [isDark, setIsDark] = useState(false)

  return (
    <div className={getThemeClassName(isDark ? 'dark' : 'light')}>
      <Button variant="primary" onClick={() => setIsDark(!isDark)}>
        {isDark ? '‚òÄÔ∏è Light' : 'üåô Dark'}
      </Button>
    </div>
  )
}
```
