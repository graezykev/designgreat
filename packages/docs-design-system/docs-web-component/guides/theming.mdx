# Theming

:::tip TL;DR
Components auto-adapt to `.dg-theme-dark` class ‚Äî no component changes needed.
:::

The library components automatically respond to theme changes through CSS variables. No component-level changes are needed.

## Quick Start

```tsx
import { useState } from 'react'
import { getThemeClassName } from '@designgreat/lib-design-token'
import { Button } from '@designgreat/lib-web-component'

function App() {
  const [isDark, setIsDark] = useState(false)

  return (
    <div className={getThemeClassName(isDark ? 'dark' : 'light')}>
      <Button onClick={() => setIsDark(!isDark)}>
        {isDark ? '‚òÄÔ∏è Light' : 'üåô Dark'}
      </Button>
      {/* All components inside auto-adapt to theme */}
    </div>
  )
}
```

## How It Works

| Theme | Class | Applied To |
|-------|-------|------------|
| Light | (none) | Default via `:root` |
| Dark | `dg-theme-dark` | Any ancestor element |

When `.dg-theme-dark` is applied to an ancestor, all components inside automatically switch to dark theme colors.
