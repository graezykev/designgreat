# Tailwind Utilities

:::tip TL;DR
Use `bg-color-*`, `text-color-*`, `p-spacing-*` classes for theme-aware custom UIs.
:::

The library includes Tailwind utilities that map directly to design tokens, allowing you to build custom UIs using the design system.

:::info Tailwind Setup
For setting up Tailwind with design tokens (CSS imports, custom variants, configuration), see the [Design Token Tailwind Guide](/design-token/guides/tailwind). This page covers **using** the pre-built utility classes.
:::

---

## Quick Example

```tsx
<div className="bg-color-background-card p-spacing-11 rounded-md">
  <h2 className="text-color-text-heading font-typography-heading-h2-font-weight">
    Custom Card
  </h2>
  <p className="text-color-text-body mt-spacing-4">
    Build with design tokens
  </p>
</div>
```

All utility classes resolve to CSS variables, so they **automatically respond to theme changes**.

---

## Utility Reference

### Colors

| Pattern | Example | Use |
|---------|---------|-----|
| `bg-color-{category}-{element}` | `bg-color-background-card` | Background colors |
| `text-color-{category}-{element}` | `text-color-text-heading` | Text colors |
| `border-color-{category}` | `border-color-border-default` | Border colors |

```tsx
<div className="bg-color-background-card text-color-text-body border-color-border-default">
  Themed container
</div>
```

### Spacing

| Pattern | Example | Use |
|---------|---------|-----|
| `p-spacing-{n}` | `p-spacing-11` | Padding all sides |
| `px-spacing-{n}` / `py-spacing-{n}` | `px-spacing-8` | Horizontal/vertical padding |
| `m-spacing-{n}` | `mt-spacing-4` | Margin (with directional variants) |

```tsx
<div className="p-spacing-11 mt-spacing-4 mb-spacing-8">
  Spaced content
</div>
```

### Typography

| Pattern | Example | Use |
|---------|---------|-----|
| `font-typography-{element}-font-weight` | `font-typography-heading-h1-font-weight` | Font weights |

```tsx
<h1 className="font-typography-heading-h1-font-weight">Bold heading</h1>
```

## Benefits

| Benefit | Without Utilities | With Utilities |
|---------|-------------------|----------------|
| **Readability** | `bg-[var(--dg-color-background-card)]` | `bg-color-background-card` |
| **Theme-aware** | Manual CSS variable handling | Automatic |
| **Consistency** | Easy to use wrong tokens | Enforces design system |

## Building Custom Components

Combine utility classes with library components:

```tsx
import { Button } from '@designgreat/lib-web-component'

function CustomCard() {
  return (
    <div className="bg-color-background-card p-spacing-11 rounded-lg shadow-md">
      <h3 className="text-color-text-heading font-typography-heading-h3-font-weight mb-spacing-4">
        Card Title
      </h3>
      <p className="text-color-text-body mb-spacing-8">
        Card description text
      </p>
      <Button variant="primary">Action</Button>
    </div>
  )
}
```

## Troubleshooting

### Styles Missing in Production

**Problem:** Some utility classes don't work after building.

**Solution:** Add the library to your Tailwind content config:

```js
// tailwind.config.js
module.exports = {
  content: [
    './src/**/*.{js,jsx,ts,tsx}',
    './node_modules/@designgreat/lib-web-component/**/*.js'
  ]
}
```
