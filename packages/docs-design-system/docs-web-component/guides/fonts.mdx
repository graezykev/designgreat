# Fonts

:::tip TL;DR
`import '@designgreat/lib-design-token/font'` before the component CSS — that's it.
:::

Fonts are **not included** in `@designgreat/lib-web-ui`. Import them from the design token package.

## Quick Start

```tsx
// In your app entry point (main.tsx, app/layout.tsx, _app.tsx)

// 1. Import fonts first
import '@designgreat/lib-design-token/font'

// 2. Then import component library
import '@designgreat/lib-web-ui/dist/lib-web-ui.css'
```

That's it! To apply the font to your app, add this to your global CSS:

```css
body {
  font-family: var(--dg-font-family-primary);
}
```

Components inherit the font from their parent elements.
