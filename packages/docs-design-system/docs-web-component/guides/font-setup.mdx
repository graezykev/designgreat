# Font Setup

Fonts are **not included** in the `@designgreat/lib-web-ui` component library. This gives you flexibility to choose your own font loading strategy.

:::tip Full Font Documentation
For comprehensive font integration including all font weights, static site generator setup, and technical details, see the [Design Token Font Integration Guide](/design-token/guides/font-integration).
:::

## Quick Start (Recommended)

Import fonts from the design token package in your application entry point:

```tsx
// In your app entry point (e.g., main.tsx, _app.tsx, app/layout.tsx)

// 1. Import fonts first
import '@designgreat/lib-web-ui-design-token/font'

// 2. Then import component library CSS
import '@designgreat/lib-web-ui/dist/lib-web-ui.css'
```

**Why this is recommended:**
- ✅ Font files are versioned with design tokens
- ✅ Includes all necessary font variants (weights 100-900, normal & italic)
- ✅ Uses optimized WOFF2 format with unicode range subsetting
- ✅ Zero configuration required

---

## Alternative: Google Fonts CDN

If you prefer not to bundle fonts, use Google Fonts:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,400;0,500;0,700;1,400&display=swap"
  rel="stylesheet"
/>
```

Then import only the component library CSS:

```tsx
import '@designgreat/lib-web-ui/dist/lib-web-ui.css'
```

## Using a Different Font

To use a different font family, override the CSS variable:

```css
:root,
.dg-theme-dark {
  --dg-font-family-primary: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
}
```

**Popular alternatives:** Inter, Open Sans, system fonts

## Troubleshooting

### Fonts Don't Load

**Problem:** Text renders in fallback font, not Roboto.

**Solution:** Import the font CSS:

```tsx
import '@designgreat/lib-web-ui-design-token/font'
```

**Common issues:**

| Problem | Solution |
|---------|----------|
| Fonts don't appear | Verify font import comes **before** library import |
| Font files 404 | Ensure `@designgreat/lib-web-ui-design-token` is installed |
| Fonts flash unstyled | This is normal with `font-display: swap`; preload critical fonts |

**Additional checks:**
- Verify font files are accessible (check Network tab in DevTools)
- Check that your bundler supports font file imports
