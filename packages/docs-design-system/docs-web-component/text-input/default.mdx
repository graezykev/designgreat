## Usage Guidelines

### When to Use
- Single-line text entry
- Email, phone, username fields
- Search inputs
- Any form requiring text input

### Best Practices
- Always include a label for accessibility
- Use placeholder for format examples, not instructions
- Keep labels concise and descriptive
- Use helper text for additional context
