# Quality Checks

Code quality tools for the documentation site.

## Lint

Check code style and potential issues:

```bash
pnpm --filter @designgreat/docs-design-system lint
```

## Type Check

Validate TypeScript types:

```bash
pnpm --filter @designgreat/docs-design-system typecheck
```

## Run All Checks

Before committing, run both:

```bash
pnpm --filter @designgreat/docs-design-system lint && \
pnpm --filter @designgreat/docs-design-system typecheck
```
