# Documentation Site Development

This section covers the development workflow, structure, and guidelines for contributing to the Design Great documentation website.

## About This Package

`@designgreat/docs-design-system` is the central documentation hub for the Design Great design system. This package provides:

- **Docusaurus Site** — Modern static site generator powered by React
- **Design Token Docs** — Integration guides for `@designgreat/lib-design-token`
- **Component Docs** — Usage guides for `@designgreat/lib-web-component`
- **Contributing Guides** — Development documentation for contributors
- **Local Search** — Client-side search with no external dependencies

## For Consumers vs Contributors

This section is for **contributors working on the documentation site itself**. If you're looking for:

- **Design token usage** — See the [Design Token Guides](/design-token/guides/overview)
- **Component usage** — See the [Web Component Guides](/web-component/guides/overview)

## Package Location

The documentation site package is located at:

```
packages/docs-design-system/
```
