# Get Started

Get the documentation site running locally on your machine.

:::info Initial Setup
Complete the [shared setup steps](/contributing#getting-started) first (clone, install, build).
:::

## Package Location

The documentation site package is located at:

```
packages/docs-design-system/
```

## Start Development Server

```bash
pnpm --filter @designgreat/docs-design-system dev
```

This command:
- Starts a local development server
- Opens your browser automatically
- Enables hot reload for most changes

The site will be available at `http://localhost:3000/designgreat/`

## Verify Setup

You should see:
- ✅ Documentation site loads in browser
- ✅ Changes to `.mdx` files hot reload
- ✅ No console errors

## Other Commands

Copy font assets to static directory (usually automatic):

```bash
pnpm --filter @designgreat/docs-design-system run copy:fonts
```
