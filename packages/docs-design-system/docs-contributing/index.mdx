# Contributing to Documentation

Welcome to the DesignGreat documentation site! This guide will help you contribute to and develop this Docusaurus-powered documentation website.

## About This Site

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator powered by React. It features:

- ✅ **Static search** - Client-side search with no external dependencies
- ✅ **Design token integration** - Consistent theming with `@designgreat/lib-web-ui`
- ✅ **Dark mode** - Automatic theme synchronization
- ✅ **Auto-deployment** - GitHub Actions workflow for continuous deployment

## What You'll Find Here

This section covers everything you need to know about working with the documentation site:

- **Getting Started** - Clone, install, and run the site locally
- **Navigation Setup** - Configure navbar and footer
- **Sidebar Configuration** - Auto-generate or manually define sidebars
- **Writing Docs** - Create and author documentation pages
- **Resources** - Additional tools and references
