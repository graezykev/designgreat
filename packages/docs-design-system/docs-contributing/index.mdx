# Contributing

Welcome to the Design Great contributor documentation! This section provides guides for working on various parts of the project.

## Getting Started {#getting-started}

All contributions start with the same setup:

### Prerequisites

- **Node.js** 22 or higher
- **PNPM** 9 or higher
- **Git**

### Clone & Install

```bash
git clone https://github.com/graezykev/designgreat.git
cd designgreat
pnpm install
```

### Build All Packages

```bash
pnpm run build
```

After this initial setup, proceed to the specific development guide for your area of contribution.
