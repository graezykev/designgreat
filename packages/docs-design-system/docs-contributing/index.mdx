## Getting Started {#getting-started}

All contributions start with the same setup:

<details>
<summary>**Prerequisites**</summary>

- **Node.js** 22 or higher
- **PNPM** 9 or higher
- **Git**

</details>

<details>
<summary>**Clone & Install**</summary>

```bash
git clone https://github.com/graezykev/designgreat.git
cd designgreat
pnpm install
```

</details>

<details>
<summary>**Build All Packages**</summary>

```bash
pnpm run build
```

</details>

After setup, proceed to the specific development guide for your area of contribution.
